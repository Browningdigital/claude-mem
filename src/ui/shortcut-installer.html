Save to Browning

Send any URL, file, photo, or PDF from your iPhone directly to the Browning Digital content pipeline. Everything is extracted, processed, and stored automatically.

Links Only

1

Create a new shortcut

Open the Shortcuts app. Tap + in the top right.

2

Name it and enable sharing

Tap the name at the top, type Save to Browning. Tap the ℹ icon (top right), toggle on Show in Share Sheet. Under share types, select anything (so it works from any app including Claude, Safari, social media, etc.).

3

Add "Get URLs from Input"

Tap Add Action. Search for Get URLs from and add it. Make sure it says Get URLs from Shortcut Input.

This step extracts clean web URLs from whatever the source app shares. Without it, some apps (like Claude, Mail, Notes) pass raw content that includes mailto: or other non-web links that break the request.
4

Add "Get Contents of URL"

Tap Add Action. Search for Get Contents of URL and tap it.

5

Set the URL

Tap the blue URL text inside the action. Clear it. Paste this:

https://content-extractor.devin-b58.workers.dev/api/share?url= Tap to copy
6

Append the extracted URL

Tap right after the = sign you just pasted. Look above the keyboard for the variables bar. Tap URLs (the output from step 3 — NOT Shortcut Input). If you don't see it, tap the {x} button above the keyboard to browse variables.

The final URL field should show the pasted text ending in = followed by a blue URLs bubble (from "Get URLs from Input"). Do not use "Shortcut Input" here — that's the raw share data which may contain mailto: links.
7

Add notification

Tap the search bar at the bottom. Type Show Notification and add it. Set the message to Saved to Browning!

8

Test it

Open Safari, tap the Share button (square with arrow), scroll down and tap Save to Browning. Also test from the Claude app, Twitter, Instagram, etc.

This version handles links AND files (photos, PDFs, videos, documents). It uses an If/Otherwise block so links go to the extractor and files get uploaded directly. Works from Safari, Claude app, social media, camera roll, Files — everything.
1

Create a new shortcut

Open the Shortcuts app. Tap + in the top right.

2

Name it and enable sharing

Tap the name at the top, type Save to Browning. Tap the ℹ icon, toggle on Show in Share Sheet. Under share types, select anything (URLs, Images, Files, etc.).

3

Add "Get URLs from Input"

Tap Add Action. Search for Get URLs from and add it. Make sure it says Get URLs from Shortcut Input.

This extracts clean web URLs from whatever the source app shares. Some apps (Claude, Mail, Notes) pass raw content containing mailto: or other non-web links. This step filters those out.
4

Add an If action

Tap Add Action, search for If and add it. Set the condition to:

If → URLs (from step 3) → has any value

This checks whether the shared item contained a web link. If yes → extract the link. If no → it's a file, upload it directly.

5

Inside If (for links)

Under the If block, add Get Contents of URL. Tap the blue URL text, clear it, and paste:

https://content-extractor.devin-b58.workers.dev/api/share?url= Tap to copy

Then tap after the = and insert the URLs variable (from step 3 — NOT Shortcut Input).

6

Inside Otherwise (for files)

Tap inside the Otherwise block. Add Get Contents of URL again. This time set it up as a POST request:

URL: Tap and paste the upload URL below.
Method: Tap "GET" and change to POST.
Request Body: Tap and change to Form.
Add field: Tap "Add new field" → choose File.
Key: Type file
Value: Tap and select Shortcut Input (the original shared item).
https://content-extractor.devin-b58.workers.dev/api/upload Tap to copy
7

Add notification (after End If)

Below the End If, add Show Notification and set the message to Saved to Browning!

8

Test it

Test with a link: share a Safari page. Test with a file: share a photo from camera roll or a PDF from Files. Test from Claude app: share a document or conversation link.
