Save to Browning

Send any URL from your iPhone directly to the Browning Digital content pipeline. Articles, videos, tweets, PDFs — all extracted, processed, and stored automatically.

Set Up Shortcut

Quick Setup (1 minute)

1

Open the Shortcuts app

Tap the + button to create a new shortcut.

2

Name it "Save to Browning"

Tap the name at the top. Then tap ⓘ → Show in Share Sheet and set it to receive URLs.

3

Add "Get Contents of URL"

Search for "Get Contents of URL" and add it. Set the URL to:

https://content-extractor.devin-b58.workers.dev/api/share?url= Tap to copy

Then append Shortcut Input to the end of the URL (tap the URL field, insert the blue "Shortcut Input" variable after =).

4

Add a notification

Add "Show Notification" and set it to: Saved to Browning!

5

Done! Test it

Open Safari, tap Share, select "Save to Browning". The page content is extracted and stored in your pipeline instantly.
